XAUUSD Predictor

Project Overview

The project involves the development of a machine learning model for predicting the XAUUSD price. The model uses input data such as open, high, low, and close prices to predict whether to buy or not. The goal of the model is to improve trading performance and returns.

Machine Learning Model

Several machine learning models were tested, including logistic regression, support vector machine (linear), and XGBoost. The chosen model is LSVM due to its high accuracy and a decent F1 score. The model can currently only trade in a long position, and if the prediction is 1, it recommends trading in a long position. The take profit (TP) is set at 3 price moves up, while the stop loss (SL) is set at 3 if the price drops.

Future Improvement and Limitation

There is room for improvement in the model. Although the LSVM model has a 70% accuracy rate, it may be improved by utilizing reinforcement learning, which is believed to be the future of trading prediction models. It is important to note that machine learning models are not foolproof and can still make incorrect predictions. Additionally, the model's accuracy may be affected by changes in market conditions, so continuous monitoring and improvement are necessary.

Applications

The project also built an application with a basic interface for users to input data, and the model will predict the result. The application also has an API for making post requests. The API is intended for use in future trading bots that can receive the current price of XAUUSD and then post a request to the API to obtain a prediction before making a trade on the MT4 and MT5 platforms. This feature could be useful for traders who want to automate their trading process.
